# Proposal

## Why

The repository is functionally ready, but it still lacks several public-repository basics: an explicit license, clear quick-start guidance, a Chinese README for the target audience, and a normalized default branch setup.

## What Changes

- Add an MIT license to the repository.
- Standardize the primary README filename to `README.md`.
- Add a Chinese README alongside the English README.
- Expand the public-facing docs with Quick Start and Requirements sections.
- Rename the local default branch from `master` to `main` so local and CI expectations align.
